That Time I Shortcut jQuery Deferred.

22 Dec 2013

Recently I had one of those moments where I tried to rely on jQuery for some promise action while working on a file uploader. But then I got snagged on IE8. Shocker.

TLDR: I ended up with an acceptable backwards compat solution that used a simple counter and deterred on jQuery deferreds.

The file uploader I was designing allowed users to upload multiple files. Now, a normal type of flow for this is fairly straightforward:

  • User uploads file(s)
  • User clicks submit
  • Code streams files wherever
  • Code executes callback in the response to submit the form

Normally you could post the entire upload object inside one request. But in this case I was using a Inc.com's FilePicker API to help us stream these files to S3. Unfortunately, this API only allowed one stream request at a time. (It's a young product). So rather than sending the file data in one ajax request, I would need to wait for an indeterminate amount of upload calls to complete before submitting the form.

jQuery has a when method for Deferred objects. So my initial idea was to take an array of these upload calls and then pass them to $.when like so:

$.when.apply($, promises).then(function(){
    form.submit();
});

But IE8 simply didn't want to play. My suspicion is that the FilePicker's API methods in my ajax calls were causing some trouble because $.when did work when I tested an isolated ajax request. Deadline looming, I had to forego a full investigation with the vendor and instead flip the script. I came up with the following (simplified for exemplification):

// Loop through each file and call stream method
$.each(arrOfFiles, function( index, file ) {
    vendorAPI.stream( file, data, function( res ){
    // Count down each time we make a call
    count--;
    // Once the count is 0, our ajax calls are done so let's submit the form
    if ( count == 0 ) {
        // Unhook dead submit binding and resubmit form
        form.submit();
    }
});

No major moral here. Just a nice bit of fun dodging bullets.

Update 6.25.14: FilePicker allows multi-uploads now. derp
